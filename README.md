# Event Book

## Project Description
### Event Book is an "event souvenir" App built in the MERN stack. After signing-in, users are able to create events and invite other users to them. When a user joins an event, he/she is able to post and store pictures and videos of the event, and also make comments.

## Technologies Used
### -Node JS.
### -React JS.
### -MongoDB.
### -Express JS.
### -oAuth 2.
### -Passport.
### -Semantic ui.
